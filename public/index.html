Daft

An uber-modern, probably wont work in your mum's browser, isomorphic-friendly JavaScript framework.


Install

$ bower install daft

Try it out

Click or tap on anything with an underline to edit.

I have a cat named Tom and a mouse named Jerry.

You'll notice if you change the name of the cat or the mouse above, it is automatically reflected below.

Hello, my name is
Hello, my name is


The Code

The above is a super simple example, and is equally as simple to implement:

Javascript
  Daft.Namespace('catAndMouse', {
    domData: {
      cat: {
        data: 'Tom'
      },
      mouse: {
        data: 'Jerry'
      }
    }
  })
HTML
  <section namespace="catAndMouse">
    <h5>
    I have a cat named <span contentEditable="true" catAndMouse-data="cat>Tom</span>
    and a mouse named <span contentEditable="true" catAndMouse-data="mouse>Jerry</span>.
    </h5>

    <div class="animal">
      <h6>Hello, my name is <span catAndMouse-data="cat"></span></h6>
      <img src="img/cat.jpg" />
    </div>

    <div class="animal">
      <h6>Hello, my name is <span catAndMouse-data="mouse"></span></h6>
      <img src="img/mouse.jpg" />
    </div>
  </section>
